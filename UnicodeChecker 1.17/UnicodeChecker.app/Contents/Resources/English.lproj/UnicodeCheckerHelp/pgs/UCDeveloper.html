UnicodeChecker

Developer Information

If you are a developer familiar with Cocoa you can develop your own String Utilities Plug-Ins and Input Plug-Ins. String Utilities must conform to the formal protocol specified in the UCUtility.framework inside the UnicodeChecker application bundle and may need to link the UCUtility.framework.

The core functionality of UnicodeChecker is available to developers through the ESUnicode.framework inside the UnicodeChecker bundle. This framework contains public header files containing most UnicodeChecker functions.

If you encounter difficulties using ESUnicode or if you want to incorporate ESUnicode into your own products please contact me. Also, if you want to develop an Input Plug-In or need help developing your plug-in, please send me an e-mail.
