UnicodeChecker Basics

UnicodeChecker is a utility to browse the Unicode character set. For any character you select, UnicodeChecker can tell you the decimal Unicode, the hexadecimal Unicode, the hexadecimal UTF-8, UTF-16 and UTF-32 code, the Unicode name (either taken from “UnicodeData.txt” or an algorithmically derived Hangul name) and lots more.

You can also go the other way round: By typing or pasting a character in the large text field or dropping a string on the main window, UnicodeChecker will present the according code point information. Furthermore, you can find code points by their corresponding Unicode name and properties.

Several utilities for converting from and to Unicode, showing individual characters in a text string, and more can be found in the “Utilities Window” which is accessible from the “File” menu.
